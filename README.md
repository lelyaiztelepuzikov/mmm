# TaskFlow — Локальный трекер задач

Минималистичный offline-first трекер задач с системой наград и штрафов.

## Возможности

- Добавление задач с названием, описанием, приоритетом и дедлайном
- Отметка задачи выполненной / снятие отметки
- Удаление задач
- Фильтрация: Все / Активные / Выполненные
- Цветовая индикация приоритета (Низкий / Средний / Высокий)
- Подсветка просроченных задач
- **Награды** за выполнение в срок (сумма + текст, например «1000 ₽ и отпуск»)
- **Штрафы** за просрочку (сумма + текст, например «без игр»)
- Финансовый баланс: заработано / штрафы / итог
- Автосохранение в localStorage — данные не теряются при перезагрузке

## Технологический стек

- HTML5, CSS3 (Flexbox, CSS Variables)
- JavaScript ES2020 (Vanilla, без фреймворков)
- Web localStorage API

## Быстрый старт

**Вариант 1 — без команд:**  
Скачай репозиторий и открой `index.html` двойным кликом в браузере.

**Вариант 2 — Live Server (VS Code):**  
Установи расширение Live Server → нажми Go Live в строке статуса.

Интернет не нужен после первой загрузки файла.

## Архитектура

Паттерн MVC в одном файле:

| Класс | Роль |
|---|---|
| `Task` | Модель задачи, атрибуты, методы |
| `TaskManager` | Бизнес-логика, коллекция задач |
| `StorageService` | Сохранение в localStorage |
| `UIController` | Рендеринг, обработка событий |
